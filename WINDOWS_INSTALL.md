# Установка для Windows (Упрощенная)

## Вариант 1: Готовый EXE файл (Рекомендуется)

### Скачать готовый файл
1. Скачайте `TelegramUnblock.exe` из релизов
2. Запустите двойным кликом
3. Готово! Приложение запущено

### Или собрать самостоятельно
1. Установите Python 3.7+ с [python.org](https://www.python.org/downloads/)
2. Запустите `build_windows.bat`
3. Найдите `TelegramUnblock.exe` в папке `dist\`
4. Запустите его

## Вариант 2: Через Python (если уже установлен)

1. Установите Python 3.7+
2. Запустите `run.bat` или `python telegram_unblock.py`

## Важно для Windows

✅ Приложение работает полностью автономно, без необходимости в дополнительных инструментах!

Встроенный SOCKS5 прокси с фрагментацией пакетов обеспечивает обход DPI.

## Быстрый старт

1. Запустите `TelegramUnblock.exe` (или `run.bat`)
2. Нажмите `1` для запуска разблокировки
3. Настройте Telegram на использование прокси:
   - Settings → Advanced → Connection type → Use custom proxy
   - Выберите SOCKS5
   - Server: 127.0.0.1
   - Port: 1080
4. Готово! Telegram должен подключиться

## Устранение проблем

### Telegram не подключается
- Убедитесь, что прокси запущен (статус должен быть "АКТИВЕН")
- Проверьте настройки прокси в Telegram (127.0.0.1:1080)
- Попробуйте изменить размер фрагмента в настройках (опция 4)
- Перезапустите приложение

### Антивирус блокирует
- Добавьте `TelegramUnblock.exe` в исключения антивируса
- Это ложное срабатывание из-за работы с сетью

### Не запускается
- Установите Visual C++ Redistributable
- Запустите от имени администратора

## Автозапуск (опционально)

1. Нажмите `Win + R`
2. Введите `shell:startup`
3. Создайте ярлык на `TelegramUnblock.exe`
4. Приложение будет запускаться с Windows

## Системные требования

- Windows 7/8/10/11
- 50 МБ свободного места
- Python 3.7+ (только для сборки)
