# Telegram Unblock - DPI Bypass Tool

Инструмент для разблокировки Telegram Desktop/Web в условиях DPI фильтрации в РФ.

## Возможности

- ✅ Обход DPI фильтрации для Telegram
- ✅ Поддержка Linux и Windows
- ✅ Простой TUI интерфейс
- ✅ Встроенный SOCKS5 прокси с фрагментацией пакетов
- ✅ Настраиваемые параметры
- ✅ Информация о подсетях и доменах Telegram
- ✅ Не требует внешних инструментов (GoodbyeDPI, Zapret и т.д.)

## Принцип работы

Приложение запускает локальный SOCKS5 прокси-сервер, который фрагментирует исходящие пакеты TLS ClientHello. Это позволяет обойти DPI фильтрацию без необходимости в root правах или внешних инструментах.

### Как это работает:
1. Запускается SOCKS5 прокси на localhost:1080
2. Первый пакет (TLS ClientHello) разбивается на мелкие фрагменты
3. Фрагменты отправляются с небольшой задержкой
4. DPI не может распознать паттерн и пропускает соединение

### Настройка Telegram:
После запуска приложения настройте Telegram:
- **Desktop**: Settings → Advanced → Connection type → Use custom proxy → SOCKS5
  - Server: 127.0.0.1
  - Port: 1080
- **Web**: Автоматически использует системный прокси
